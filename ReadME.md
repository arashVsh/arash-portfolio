## Selected Projects

### PaperWise AI

A deployed research-paper question-answering application with citation-grounded answers, paper summaries, reviewer critique mode, multi-paper comparison, and PDF answer export.

### PromptShield

A machine learning framework for detecting adversarial images in vision-language models using representation-level features.

### AttackBench

A desktop application for generating, running, and visualizing adversarial attacks on image classification models, built with JavaFX and a Python FastAPI backend.
